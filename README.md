This repository contains the core analytical scripts used in our study investigating the genetic and multi-omics determinants of multiple sclerosis (MS) progression.

## Overview of analyses

The repository includes scripts for:

- **Proteome-wide association study (PWAS)** using genetically predicted protein abundance  
- **Summary-based Mendelian Randomization (SMR)** integrating GWAS and pQTL data  
- **Transcriptome-wide association study (TWAS)** using the FUSION framework  
- **Two-sample Mendelian Randomization (MR)** for causal inference  
- **Single-cell RNA-seq differential expression analysis** to investigate cell-type specificity
